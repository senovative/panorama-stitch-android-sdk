# Security Policy

## Supported Versions

We currently only support the latest version of the SDK for security updates. 

## Reporting a Vulnerability

If you discover a security vulnerability within this SDK, please DO NOT open a public issue. Instead, please send an email to the repository maintainer. All security vulnerabilities will be promptly addressed.

We will acknowledge receipt of your vulnerability report and strive to send you regular updates about our progress. If you report a vulnerability, you will be credited in the release notes when the fix is published.
